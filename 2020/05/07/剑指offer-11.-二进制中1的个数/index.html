[剑指Offer] 11. 二进制中1的个数

May 7, 2020 #位运算符

1.题目描述

输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

题目链接：https://www.nowcoder.com/practice/8ee967e43c2c4ec193b040ea7fbb10b8?tpId=13&tqId=11164&tPage=1&rp=1&ru=%2Fta%2Fcoding-interviews&qru=%2Fta%2Fcoding-interviews%2Fquestion-ranking%3E

2.解题思路

略。

public class Solution {
    public int NumberOf1(int n) {
        int count=0;
        while(n != 0){
            n &= n-1;  //得到二进制去掉最右边1后剩下的二进制数
            n >>>= 1;
            count++;
        }
        return count;
    }
}

运行时间：13ms

占用内存：9108k

3.位运算符相关

3.1 常见位运算&和&&

(1). &叫做位的与运算符

n &= n-1 -> 得到二进制去掉最右边1后剩下的二进制数（例如当n=00010101，经过n &= n-1后得到00010100）；

n &= (-n) -> 得到二进制只剩最右边1的二进制数（例如当n=00010101，经过 n &= (-n) 后得到00000001）。

(2). &&叫做逻辑与运算符，又叫短路运算符。当左边条件执行结果为false时，短路运算符的右边条件不会执行，效率相对&效率更快。

3.2 常见位运算^（异或：不同为1，相同为0）

3.3 常见位运算»和»>

(1). » （带符号右移）：正数右移高位补0，负数右移高位补1 。

(2). »> （无符号右移）：无论正数还是负数，高位通通补0。

对于正数而言，»和»>没有区别。
