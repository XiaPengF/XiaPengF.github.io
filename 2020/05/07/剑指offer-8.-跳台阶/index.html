[剑指Offer] 8. 跳台阶

May 7, 2020 #递归、动态规划

1.题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

题目链接： https://www.nowcoder.com/practice/8c82a5b80378478f9484d87d1c5f12a4?tpId=13&tqId=11161&tPage=1&rp=1&ru=%2Fta%2Fcoding-interviews&qru=%2Fta%2Fcoding-interviews%2Fquestion-ranking%3E

2.解题思路

此题其实就是一个斐波那契数列。

当 $n=1、2$时：$f(n)=1、2$ ；

当 $n>=3$时： $f(n) = f(n-1)+f(n-2) $

2.1.使用递归

public class Solution {
    public int JumpFloor(int target) {
        if(target<3){
            return target;
        }
        return JumpFloor(target-1)+JumpFloor(target-2);
    }
}

运行时间：383ms

占用内存：9416k

2.2.使用动态规划(借助数组)

public class Solution {
    public int JumpFloor(int target) {
        if(target<3){
            return target;
        }
        int[] dp = new int[target+1];
        dp[0]=0;dp[1]=1;dp[2]=2;
        for(int i=3;i<target+1;i++){
            dp[i] = dp[i-1]+dp[i-2];
        }
        return dp[target];
    }
}

运行时间：16ms

占用内存：9292k

2.3.使用动态规划(不借助数组)

public class Solution {
    public int JumpFloor(int target) {
        if(target<3){
            return target;
        }
        int prev = 1, curr = 2;
        for(int i=3;i<=target;i++){
            int sum  = prev + curr;
            prev = curr;
            curr = sum;
        }
        return curr;
    }
}

运行时间：13ms

占用内存：9424k
